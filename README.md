## Environments
Create and set .env
```bash
#Mandatory
AMADEUS_API_KEY= #Amadeus api key
AMADEUS_API_SECRET= #Amadeus api secret
```

```bash
#Optional
NODE_PORT= #default 3000,
NODE_ENV= #default 'develop'
NODE_URL= #default 'localhost:3000'
```
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
